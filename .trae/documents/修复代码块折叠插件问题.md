## 问题修复计划

### 1. 修复折叠功能问题
- **问题**：CSS 选择器 `.code-block-folded code.code-content-folded` 不正确，导致折叠样式不生效
- **修复**：修改 CSS 选择器为 `.code-block-folded > code` 或调整 class 应用方式
- **新增**：让点击整个 header 也能触发折叠/展开

### 2. 隐藏 Obsidian 自带复制按钮
- **问题**：Obsidian 自带的 `copy-code-button` 类按钮与插件按钮重叠
- **修复**：在 styles.css 中添加 CSS 隐藏 `.copy-code-button`

### 3. 添加语言显示
- **问题**：header 中没有显示代码块的语言
- **修复**：在 main.ts 中：
  - 从 code 元素的 class 中提取语言（如 `language-javascript` → `javascript`）
  - 在 header 左侧添加语言标签显示
  - 如果没有设置语言，显示 "Code-Block"

### 修改文件
1. `styles.css` - 修复折叠样式、隐藏自带复制按钮、添加语言标签样式
2. `src/main.ts` - 添加语言提取逻辑、header 点击折叠功能
